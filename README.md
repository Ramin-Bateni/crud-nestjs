## Description

Task for interview

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# bdd tests
$ npm run test:bdd
```

## Deployment

```bash
# run the dockerized version of the app
docker compose up -d
```
